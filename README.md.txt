# yt-shorts-bot

🤖 Bot automatico per creare e pubblicare YouTube Shorts su giochi e stampa 3D.

## 🧰 Funzioni:
- Genera testo con modello locale
- Sintetizza voce in italiano
- Monta video verticale
- Carica su YouTube
- Manda link su Telegram

## ⚙️ Installazione
```bash
chmod +x install.sh
./install.sh
